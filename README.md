# 📱 Social Media Feed Aggregator

## 📌 Objective
This project is a simple Social Media Feed Aggregator that combines data from multiple platforms into a unified view.

---

## 🚀 Features

- Aggregates posts from multiple platforms (e.g., YouTube, Twitter/X)
- Uses JSON data (API or sample data)
- Normalizes data into a common structure
- Displays posts in a clean UI (card/list view)

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- JSON  

---


## 🔄 Data Format

Each post is normalized into:

- Platform  
- Title  
- Image  
- Date  
- Link
